Welcome to FGC, a non-profit for Free Learning on Real Life Development of Dynamic Web Applications Using LAMP, with PHP and open-source software.

The web applications streamline the procedures in refurbishing computers and associated items, which go through the process of intake, triage, build, quality check, inventory, sales, scrapping/recycling, and financial accounting. This is supported by a time management system which keeps track of volunteer orientation, training, activities done and certificates, contributed hours, and earned/redeemed reward points. Future modules will include monitoring of free classes on introduction to computers, programming languages, etc.
